# Aída Maestro Photography

A clean one-page landing site for Aída Maestro, a professional editorial and commercial photographer.

## What this site includes

- One landing page with hero, about, work, process, and contact sections.
- Direct email booking via `y@criptec.io`.
- No additional pages, no blog, no language switcher, no theme switcher, no analytics dashboard.
- Refined grey palette and calm editorial typography.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
